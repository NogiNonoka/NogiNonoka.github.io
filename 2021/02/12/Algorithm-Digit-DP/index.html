Algorithm: Digit DP (DFS Memory Search)

Created2021-02-12|Updated2021-02-13|Algorithm
|Post View:

解决问题

数位 DP 通常用于解决区间 [L, R][L,\ R][L, R] 内符合条件的数的个数。

算法思想

数位 DP 的实现主要有两种：

  • 递推 构建转移方程
  • 记忆化搜索

其中，记忆化搜索更易实现且能解决更多问题，本文主要介绍记忆化搜索的实现方式。

算法实现

基于 DFS 的记忆化搜索，主要维护当前位数、条件、数位上限等参数。

  • 通过待求区间上限，计算 limlimlim 得到数位上限与 pospospos 最大值；
  • 递归更新当前条件值 / 数位值等，判断是否处于上限；
  • 更新 DP 数组。

链接：P2657

Problem

Analysis
  • 前缀和思想，处理区间 [1, b]−[1, a−1][1,\ b] - [1,\ a - 1][1, b]−[1, a−1]。

2020 ICPC Jinan L - Bit Sequence

链接：Bit Sequence

Problem
  • 给定长度为 m (1≤m≤100)m \ (1 \le m \le 100)m (1≤m≤100) 的二进制序列 aaa，aia_iai​ 表示 x+ix + ix+i 中 111 个数的奇偶性；
  • 询问在区间 [0, l] (0≤l≤1018)[0,\ l] \ (0 \le l \le 10^{18})[0, l] (0≤l≤1018) 内，满足条件的 xxx 的个数。
Analysis
  • 注意到 m≤100m \le 100m≤100，故产生变化的位数在最后 8 bit8 \ bit8 bit；
  • 同时，后 8 bit8 \ bit8 bit 可能产生进位对前连续的 111 个数的奇偶性可能产生影响；
  • 故，影响 aia_iai​ 的所有可能条件为：
    • 后 8 bit8 \ bit8 bit 的中 111 个数的奇偶性；
    • 除后 8 bit8 \ bit8 bit 剩余 111 个数的奇偶性；
    • 产生进位，后 8 bit8 \ bit8 bit 前连续的 111 个数的奇偶性。
