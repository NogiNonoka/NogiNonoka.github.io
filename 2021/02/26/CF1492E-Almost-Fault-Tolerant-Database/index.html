CF1492E Almost Fault-Tolerant Database

Created2021-02-26|Updated2021-02-26|Contest
|Post View:
Problem
  • 给定 nnn 个长度为 mmm 的数组（2≤n, 2≤n⋅m≤2500002 \le n ,\ 2 \le n \cdot m \le 2500002≤n, 2≤n⋅m≤250000）；
  • 询问是否存在长度为 mmm 的数组，使得原数组中任一一组数据与所求数组不同的元素个数 ≤2\le 2≤2。
Analysis
  • 思维 贪心 模拟
  • 设所求数组为第一组数据，与其他组数据比较，求不同元素个数 difdifdif；
    • ∀ dif≤2\forall \ dif \le 2∀ dif≤2，第一组数据合题意；
    • ∃ dif>4\exist \ dif > 4∃ dif>4，不存在；
  • 对于 3≤dif≤43 \le dif \le 43≤dif≤4，
    • 若 difi=4dif_i = 4difi​=4，则至少修改其中两个，枚举所选的两个数并判断，若枚举均不合法，则不存在；
    • 若 difi=3dif_i = 3difi​=3，则可能修改一个或两个，
      • 考虑修改一个的情况，若修改后新数组 newa1newa_1newa1​ 与其余数组 aja_jaj​ 仍有 newdifj=3newdif_j = 3newdifj​=3，则需考虑 newa1newa_1newa1​ 与 aja_jaj​ 并再枚举修改一个并判断；
      • 枚举修改两个的情况，并判断。

本题难点在于处理 dif=3dif = 3dif=3 时，修改两个元素但修改值不在同一组数中的情况。
