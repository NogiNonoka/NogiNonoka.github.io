1487C - Minimum Ties

Problem
  • nnn 只队伍踢足球比赛，胜者积三分，败者不积分，平局双方积一分；
  • 每两支队伍踢一场比赛，各队最终积分相等，构造平局最少的比赛结果。
Analysis
  • 思维 贪心 图论
  • 偶数支球队，每支踢 n−1n - 1n−1 场，胜负各半；
  • 奇数支球队，平局一场，剩余胜负各半，构造时，贪心策略：能赢就赢。

1487D - Pythagorean Triples

Problem
  • 正整数范围内，能构成直角三角形的三个整数且满足 c=a2−bc = a^2 - bc=a2−b 的三元组有多少个。
Analysis
  • 数学
  • c=a2−bc = a^2 - bc=a2−b 且 c2=a2+b2c^2 = a^2 + b^2c2=a2+b2 可得 (c+b)(c−b−1)=0(c + b)(c - b - 1) = 0(c+b)(c−b−1)=0；
  • 故，c=b+1, a2=2b+1<nc = b + 1 ,\ a^2 = 2b + 1 < \sqrt nc=b+1, a2=2b+1<n​，枚举 [1, n][1 ,\ \sqrt n][1, n​]。
