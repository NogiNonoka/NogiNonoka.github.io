  • 线性同余方程
    • 形如：ax≡c (mod b)⇔ax+by=cax \equiv c \ (mod \ b) \Leftrightarrow ax + by = cax≡c (mod b)⇔ax+by=c
    • 方程有解的条件：gcd(a,b)∣cgcd(a,b) | cgcd(a,b)∣c；
  • 扩展欧几里得
    • 解不定方程 ax+by=gcd(a,b)ax + by = gcd(a, b)ax+by=gcd(a,b)；
